How to review
  1. Walk the interior page set in this order: article → category → company → ticker → person → sector.
  2. Check consistency at the chrome level: sticky pill-nav header, breadcrumb, ink footer. Each page inherits them intentionally.
  3. Flag any module that feels like it belongs on a different direction — this is the last cheap moment to adjust.
  4. Once signed off, this direction becomes the visual spec that Claude Code uses to build the live Next.js components.
